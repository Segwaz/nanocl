# NAME

nanocl

# SYNOPSIS

**nanocl** \[**-s**\|**\--state-location**\] \[**-y**\|**\--yes**\]
\[**-h**\|**\--help**\] \[*ARGS*\]

# DESCRIPTION

# OPTIONS

**-s**, **\--state-location**=*STATE_LOCATION*

:   Path or Url to the Statefile

**-y**, **\--yes**

:   Skip the confirmation prompt

**-h**, **\--help**

:   Print help

\[*ARGS*\]

:   Additional arguments to pass to the file
